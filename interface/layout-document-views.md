* **?** only displayed if exist
* **-** line
* **==** dictionary key,value
* **_=** dictionary value
* **=_** dictionary Key
* **=** list
* **=,** list joined by ,
* **t** translated
* **l** link
* **B** text as badge
* **b** with badge
